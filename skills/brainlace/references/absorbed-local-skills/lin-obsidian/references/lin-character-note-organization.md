# LIN character/backstory notes

Use this when a note about LIN, another character, or a persona starts mixing two layers:

1. **entry note / profile note** — short, easy to skim, the current basic picture
2. **backstory / origin / source-image note** — heavier lore, naming origin, creation circumstances, image interpretation, comparisons with adjacent characters

## Recommended split

Keep the main note light:
- one-paragraph summary
- current traits / role / reading guide
- at most a short origin teaser
- explicit wikilink to the deeper note

Move these into a separate note:
- naming etymology
- creation-day chronology
- source-image interpretation
- comparisons like `A` versus `B`
- long origin stories, lore temperature, symbolic meaning

## Why this helps

- the entry note stays readable as an overview
- the backstory can grow without bloating the main note
- future edits to lore do not destabilize the canonical short intro
- related notes like relationship notes can link to the deep note without repeating the whole story

## Pattern used here

For LIN notes in this workspace:
- `LIN.md` = lightweight entry note
- `バックボーンと原型画像.md` = origin, LIMBO contrast, source-image reading
- `INDEX.md` should link both

## Quick checklist

- create the deep note first
- move the long lore there, not just duplicate it
- leave a short pointer section in the entry note
- update the folder index so both notes are discoverable
- if the conversation included image interpretation, save the distilled reading as prose, not only as a raw screenshot reference
